ドット絵PC操作ミッション

見本をよく見て、ドット絵を完成させよう。

ルール

  • 練習モードでは、16×16マスで「1」がある場所を黒くぬろう。
  • 上級モードでは、32×32マスで「白色・薄灰色・濃灰色・黒色」を合わせよう。
  • 色をまちがえたら、同じマスをもう一度クリックして変えられます。
  • ドラッグすると、続けてぬれます。

少ない色でも、組み合わせると絵を表せます。
