# AGENTS.md

このリポジトリで作業するAIエージェント向け方針です。

- 授業品質（安全・迷いにくさ・復帰しやすさ）を最優先にしてください。
- 個人情報の収集・保存・送信を追加しないでください。
- 解析タグ、トラッカー、外部APIなどの外部サービスを追加しないでください。
- 明確な理由がない限り、フレームワークやビルドツールを追加しないでください。
- 生徒向けUIは日本語で統一してください。
- GitHub Pagesで動く静的構成（HTML/CSS/JavaScript）を維持してください。
- 任意テキスト・絵文字の変換機能は将来拡張とし、明示要望がない限りMVPに含めないでください。
- 明示要望がない限り、自動コピー機能ボタンを復活させないでください。
- 明示要望がない限り、0/1データの可視テキストエリアを復活させないでください。
- 手動の Ctrl+C / Ctrl+V 練習はコア要件として扱ってください。
- 明示要望がない限り、Undo / 「1つ戻す」を復活させないでください。授業試行で、繰り返しクリックが多くUndoは有効に働かなかったためです。
- 明示要望がない限り、練習モード・上級モード両方のBMP保存機能を維持してください。
- 上級モードのBMPは、Windows Paint互換性のため、原則として4bppインデックスBMPと16エントリのパレットを維持してください。変更する場合は理由を明記してください。
